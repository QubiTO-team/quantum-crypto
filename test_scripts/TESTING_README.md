# How to use the testing suite?
1* - Open the make_circuit.py file and configure it to create the circuit you need.
2* - Run the make_circuit.py file to save the circuit.
3 - Run the gen_test_qbits.py file by specifying the number of shots and the circuit to use.
(Optional) - If needed, use rename_circuit.py by specifying the name of the circuit to rename and the new name.
This will update all the .json files with the proper circuit.

*: Steps 1 and 2 can be skipped if you need to use a 1-Bit Hadamard circuit (this is already added in the circuits folder)
